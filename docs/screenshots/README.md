# MyMoney Notes Screenshots

This folder contains screenshots of the MyMoney Notes application.

## Screenshots

1. **app_thumbnail.png** - App thumbnail for presentation
2. **home_screen.png** - Dashboard showing balance, income, and expense
3. **add_transaction.png** - Add transaction form
4. **transaction_list.png** - Transaction history with filters
5. **charts_screen.png** - Analytics with pie chart

## How to Add Screenshots

1. Run the app on emulator or device
2. Navigate to each screen
3. Take screenshots:
   - Windows: Press `Win + Shift + S` or use Android Studio's built-in screenshot tool
   - Mac: Press `Cmd + Shift + 4`
4. Save images to this folder
5. Rename files according to the list above
